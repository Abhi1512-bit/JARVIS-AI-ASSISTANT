# Jarvis AI Assistant

A Python-based voice assistant that uses Speech Recognition, Text-to-Speech, NewsAPI, and Google Gemini AI to perform tasks such as opening websites, playing music, reading news headlines, and answering user queries.

## Features

* Voice command recognition
* Wake word activation ("Jarvis")
* Open websites like Google, YouTube, LinkedIn, and Facebook
* Play music from a custom music library
* Read latest news headlines using NewsAPI
* AI-powered responses using Google Gemini
* Text-to-Speech interaction

## Technologies Used

* Python
* SpeechRecognition
* Pyttsx3
* Requests
* Google Gemini API
* NewsAPI

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/Jarvis-AI-Assistant.git
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Add your API keys.

4. Run the project:

   ```bash
   python main.py
   ```
